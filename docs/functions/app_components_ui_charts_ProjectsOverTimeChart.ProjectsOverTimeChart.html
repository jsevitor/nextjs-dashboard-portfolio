      ProjectsOverTimeChart Component

      Componente responsável por exibir um gráfico de linha (line chart) que mostra a quantidade de projetos ao longo do tempo, agrupados por mês. A quantidade de projetos é contada para cada mês com base na data de criação dos projetos. Enquanto os dados estão sendo carregados, um indicador de carregamento é exibido. Caso não haja dados disponíveis, uma mensagem é apresentada.

      ▸ Responsabilidade

      • Contabilizar a quantidade de projetos criados a cada mês
      • Exibir um gráfico de linha mostrando a evolução dos projetos ao longo do tempo
      • Exibir um loader enquanto os dados são carregados
      • Exibir uma mensagem caso não existam dados disponíveis

      Returns Element

      Componente que exibe um gráfico de linha com a quantidade de projetos por mês
